# 頻出コード

・Laravel起動
```
[Ubuntu]
~/book-data$ sail up -d
```

・Laravel終了
```
[Ubuntu]
~/book-data$ sail stop
```
・Tailwind CSS起動
```
[Ubuntu]
~/book-data$ sail npm run dev
```

・有効ルートの表示
```
[Ubuntu]
~/book-data$ sail artisan route:list
```
